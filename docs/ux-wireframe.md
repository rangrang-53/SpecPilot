# SpecPilot UX 와이어프레임

## 1. 초기 화면 (Initial Screen)

![Initial Screen](../wireframes/01-initial-screen.svg)

**구성 요소:**
- 헤더: SpecPilot 로고 및 타이틀
- 사이드바: 진행 상태, 세션 리셋 버튼
- 메인 영역: 환영 메시지 및 입력 필드
- 입력창: 초기 요구사항 입력

---

## 2. 질문-응답 화면 (Q&A Screen)

![Q&A Screen](../wireframes/02-qa-screen.svg)

**구성 요소:**
- 채팅 히스토리: 사용자와 AI의 대화 기록
- Consultant 질문 카드: AI가 생성한 질문 목록
- 입력창: 사용자 답변 입력
- 진행 상태 표시: 현재 반복 횟수

---

## 3. 문서 생성 완료 화면 (Completion Screen)

![Completion Screen](../wireframes/03-completion-screen.svg)

**구성 요소:**
- 완료 알림: SRS 생성 완료 메시지
- 문서 미리보기: 생성된 SRS 내용 요약
- 다운로드 버튼: Markdown/JSON 형식 선택
- 새 세션 시작 버튼

---

## 4. 사이드바 상세 (Sidebar Detail)

![Sidebar Detail](../wireframes/04-sidebar.svg)

**구성 요소:**
- 프로젝트 정보
- 진행 단계 인디케이터
- Iteration 카운터
- 세션 리셋 버튼
- 도움말 링크
